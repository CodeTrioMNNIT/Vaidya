VAIDYA 

This project is a Full Stack Doctor Appointment Booking System built using the MERN stack. It allows users to book, manage, and pay for doctor appointments online.  

Features  
1. Patient Dashboard  
- Register and log in securely.  
- Book appointments with doctors based on availability.  
- Manage and view booked appointments.  

2. Doctor Dashboard  
- Secure login for doctors.  
- View and manage appointments.  
- Check earnings and update profiles.  

3. Admin Dashboard  
- Manage all appointments.  
- Oversee and update doctor profiles.  

4. Online Payment Integration  
- Seamless payment gateway integration for appointment fees.  

Tech Stack  
- Frontend: React.js  
- Backend: Node.js, Express.js  
- Database: MongoDB
